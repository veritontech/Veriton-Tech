DEVELOPMENT WORKFLOW
====================
1. npm install           (Run once, or when dependencies change)
2. npm run dev           (Start dev server)
3. Edit files in src/    (Auto-reloads browser)
4. Check terminal        (For errors)
5. Check browser F12     (For runtime errors)
6. Ctrl+C                (Stop server when done)


DEPLOYMENT WORKFLOW
===================
1. npm run build         (Build for production)
2. Check dist/ folder    (Verify files created)
3. npm run preview       (Test production build locally)
4. Upload dist/ folder   (To your hosting service)
