## File Naming Conventions

- **Components:** PascalCase (e.g., `HeroSection.tsx`)
- **Utilities:** camelCase (e.g., `utils.ts`)
- **CSS files:** kebab-case or lowercase (e.g., `theme.css`)
- **Config files:** lowercase with extensions (e.g., `vite.config.ts`)

## Import Aliases

The `@` symbol is an alias for the `/src` directory:

```tsx
// Instead of:
import { Header } from '../../../components/layout/Header';

// You can use:
import { Header } from '@/app/components/layout/Header';
```

**Configured in:** `vite.config.ts` and `tsconfig.json`

## Common Editing Tasks

### Adding a New Page

1. Create file in `/src/app/pages/NewPage.tsx`
2. Create component:
```tsx
export function NewPage() {
  return <div>Your content</div>;
}
```
3. Import in `/src/app/App.tsx`
4. Add to navigation in Header.tsx

### Adding a New Component

1. Create file in appropriate folder (e.g., `/src/app/components/home/NewSection.tsx`)
2. Export component:
```tsx
export function NewSection() {
  return <div>Your section</div>;
}
```
3. Import in the page where you want to use it
4. Add to page JSX

### Changing Colors

1. Open `/src/styles/theme.css`
2. Modify CSS custom properties:
```css
:root {
  --color-primary: blue;  /* Change this */
}
```
3. Or use Tailwind classes in components:
```tsx
<div className="bg-blue-600">
```

### Adding Images

1. Create `/public/images/` folder if it doesn't exist
2. Add your images there
3. Reference in components:
```tsx
<img src="/images/your-image.jpg" alt="Description" />
```

## Build Process

1. **Development:** `npm run dev`
   - Vite starts dev server
   - Watches for file changes
   - Hot reloads on save
   - Serves at http://localhost:5173

2. **Production:** `npm run build`
   - TypeScript compiles to JavaScript
   - Vite bundles all files
   - Tailwind CSS purges unused styles
   - Output goes to `/dist` folder
   - Files are minified and optimized
